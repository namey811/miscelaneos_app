# MISCELANEOS APP

A new Flutter project en donde aprenderemos a otorgar permisos de camara, contactos, almacenamiento.

## Dependencias de la aplicacion

Instalar las siguientes dependencias del proyecto:

```
google_fonts
go_router

```
